# Flask Memo App

## 🔎概要
Flaskで作成したシンプルなメモアプリです。
メモの管理を「見た目」と「操作感」にこだわって実装しました。

## ✅できること
- メモの追加・編集・削除
- 完了／未完了の切り替え
- 完了メモを元に戻す
- タグ付け（重要・仕事・買い物・急ぎ など）
- タグ別検索
- キーワード検索

## 🎨工夫したポイント
- ピンクを基調にした、可愛すぎないシンプルなデザイン
- 完了・削除ボタンにホバーエフェクト
- タグごとに色分けして視認性を向上
- 完了メモは全体を薄く表示し、状態が一目でわかる
- 完了／戻す操作時にアニメーションを追加
- 完了メモにカーソルを合わせると色が戻る演出

## 🛠️使用技術
- Python
- Flask
- HRML / CSS
- JavaScript
- Jinja2

## 🚀学んだこと
- Flaskを使ったWebアプリの基本構造
- 状態管理（完了／未完了）の実装
- UI／UXを意識したデザインとアニメーション
- バグ修正・原因の切り分け
